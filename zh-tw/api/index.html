API 文件

COSCUP 2015 網站自 2018/08/20 起移至 GitHub Pages 管理，故不再提供 JSON-P API 供第三方程式取用。僅提供靜態 JSON 檔案；各 URL 皆會輸出 Access-Control-Allow-Origin: * HTTP 標頭。

舊的 API 文件頁面可以在 這裡 找到

議程 (program)

  • URL: https://coscup.org/2015/api/program.json 縮排顯示。

贊助單位 (sponsors)

  • URL: https://coscup.org/2015/api/sponsors.json 縮排顯示。

COSCUP 2015 的贊助商資訊。程式應內建各贊助等級的順序，各等級內的贊助單位順序在列出時應保存；在只能顯示一個贊助單位的場合，要顯示哪個贊助單位的機制必須使用加權隨機的方式挑選，加權比率如下：

diamond:gold:silver:bronze:media = 10:5:2:1:0

此演算法的 Javascript 版本在 COSCUP 2014 手機版網頁有實作，您可以參考此處位於 mobileSponsorLogo() 的程式碼。

程式至少需每日更新贊助單位資訊。
